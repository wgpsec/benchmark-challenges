PDF Generation Service

Generate PDFs from any URL. Enter a URL below to fetch and convert to PDF.

API Endpoints

  • POST /api/generate-pdf - Generate PDF from URL
  • POST /api/fetch - Fetch URL content (debug)
  • POST /api/proxy - Full proxy with custom headers

Example

curl -X POST http://localhost:3010/api/generate-pdf \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}'
