Template Injection Lab

Welcome to the Server-Side Template Injection (SSTI) demonstration application.

Available Endpoints

1. Greeting Page (GET)

Personalized greeting using your name.

/greet?name=World

Try: /greet?name={{7*7}}

2. Template Preview (POST)

Preview Jinja2 templates in real-time.

/preview

3. Error Page (GET)

Custom error messages.

/error?message=Something went wrong

4. Safe Greeting (GET)

Properly implemented (not vulnerable).

/greet_safe?name={{7*7}}

Security Notice: This application contains intentional SSTI vulnerabilities for educational and security research purposes. Do not deploy in production.

SSTI Quick Reference

Detection

{{7*7}}          -> 49
{{'a'*5}}        -> aaaaa
{{config}}       -> Flask config

Exploitation Chain

{{''.__class__}}                              -> <class 'str'>
{{''.__class__.__mro__}}                      -> Class hierarchy
{{''.__class__.__mro__[1].__subclasses__()}}  -> All classes
